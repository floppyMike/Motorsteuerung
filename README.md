# Motorsteuerung
## Grober Übersicht
![Basic Graph](images/basic.dot.svg)
`Setup` dient zur Initialisierung des Programms und wird beim Start des Programms aufgerufen. `Loop` wird nach `Setup` aufgerufen und wird während der Programmzeit wiederholt ausgeführt. `Loop` ist für den Betrieb des Elektroautos zuständig.
